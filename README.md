# Todo App

## About:
Simple Todo application for code show.

## Requirements:
```
docker >= "3.1"
php >= "7.2"
node.js >= "10.16.0"
```

## Install:
```
cd backend && composer install
cd frontend && npm install
```

## Tests:
```
cd backend && composer test
```

## Run:
1. `docker-compose up`
2. `cd backend && composer serve`
3. `cd frontend && npm run serve`
